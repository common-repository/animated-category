Animated Category plugin easily is an high customizable and responsive widget for displaying category. This plugin also customize widget color. Count category on off facilities also included.


[Demo Plugin](https://bdcodemaker.com/demo/plugin/animcat/)


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'anim-cat.php' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Go your Appearance option and select widget option.
3. Drag and drop Animated widget where you need.
